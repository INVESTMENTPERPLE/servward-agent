Live monitoring

Health & disks (SMART)

Push alerts

Notifies you when a server goes down or a threshold you set is crossed.

Remote control

Restart services, run commands, set volume, sleep or kill processes.

One tap in Settings. Mac and Linux, no terminal needed.

Multi-server + QR

Add all your servers by scanning a QR. Switch between them instantly.

🧩

Widget & Live Activity

Status on your Home Screen and down alerts in the Dynamic Island.

100% private

How it works

Set up in minutes.

1

Install the agent

One command on your server, Mac or Linux. Keeps token and settings on update.

2

Connect it

Over Tailscale (easiest) or a Cloudflare tunnel, using the token the installer prints.

3

Control from your iPhone

Metrics, health, alerts and actions — from anywhere.

curl -fsSL https://raw.githubusercontent.com/INVESTMENTPERPLE/servward-agent/main/install.sh | bash
