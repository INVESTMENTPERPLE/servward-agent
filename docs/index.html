Dudas rápidas

Preguntas frecuentes

¿Tengo que abrir puertos en el router?
No. Con Tailscale o un túnel de Cloudflare la conexión sale de tu servidor hacia fuera: nada de port-forwarding ni de exponer tu IP a internet.
¿Qué necesita mi servidor?
Un Mac (launchd) o un Linux con systemd, con git y Python 3. El instalador de un comando monta el broker y el agente, y te imprime el QR para conectar la app.
¿Mis datos pasan por servidores de terceros?
No. La app habla directamente con tu máquina. Solo las notificaciones push viajan por APNs (Apple), sin contenido sensible.
¿Es de pago?
El agente es open source (MIT) y gratuito para siempre. La app está ahora en beta gratuita por TestFlight.
¿Cuántos servidores puedo controlar?
Varios: añádelos escaneando un QR y cambia entre ellos al instante. Mac y Linux mezclados, sin problema.
¿Y si mi servidor se cae?
Recibes una alerta push al momento y una Live Activity con el tiempo que lleva caído y un botón para reintentar la conexión.
¿Y si no me convence?
Un comando y desaparece: ntfyctl uninstall para y elimina los servicios, el código y la configuración del servidor.

Quick answers

Frequently asked questions

Do I need to open ports on my router?
No. With Tailscale or a Cloudflare tunnel the connection goes outbound from your server: no port-forwarding, no exposing your IP to the internet.
What does my server need?
A Mac (launchd) or a Linux box with systemd, plus git and Python 3. The one-command installer sets up the broker and the agent, and prints the QR to connect the app.
Does my data go through third-party servers?
No. The app talks directly to your machine. Only push notifications travel through Apple's APNs, with no sensitive content.
Is it paid?
The agent is open source (MIT) and free forever. The app is currently in a free TestFlight beta.
How many servers can I control?
Several: add them by scanning a QR and switch instantly. Mac and Linux mixed, no problem.
What if my server goes down?
You get an instant push alert and a Live Activity showing how long it's been down, with a retry button.
What if it's not for me?
One command and it's gone: ntfyctl uninstall stops and removes the services, code and configuration from your server.
