Get started

Set up in minutes.

Download the Mac agent from GitHub, run the installer, and connect using Tailscale or Cloudflare Tunnel. Then configure the iOS app with your server URL and token.

git clone https://github.com/INVESTMENTPERPLE/NFTYcontrol.git
cd NFTYcontrol
bash setup_launchagent.command

Option A

Tailscale

Install Tailscale on your Mac and iPhone. Use the 100.x.x.x address as your server URL. Zero config.

Option B

Cloudflare Tunnel

Expose your Mac with a public domain. No extra app needed on the client side.
